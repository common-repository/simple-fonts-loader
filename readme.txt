== Description ==

Un simple plugin qui permet d'activer des polices d'écritures de Google Fonts et de mettre les @font-face sur le site.
Aucun code supplémentaire nécessaire. Aucune requête vers Google Fonts. Tout en local.
Il vous suffit juste d'activer les polices qui vous intéresse, vous pouvez copier directement les propriétés font-family, font-weight et font-style, et c'est parti.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-fonts-loader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Fonts screen to enable or disable the fonts
4. Click on copy, next to the font name, to copy font-family property
